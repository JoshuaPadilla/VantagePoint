# VantagePoint

> Prediction of Student Performance in the Entrance Examination using Classification Techniques

VantagePoint is a full-stack web application built as a thesis project. It uses a trained **Random Forest Classifier** to predict a student's expected performance category in an entrance examination based on demographic, academic, and socio-economic inputs. The system delivers predictions through a clean, interactive UI backed by a Python ML service and a TypeScript REST API.

---

## Features

- Fill out a structured form with student background data
- Receive an instant prediction: **Excellent**, **Very Good**, **Good**, or **Average**
- View a confidence score and animated gauge for each prediction
- Blog section documenting the methodology and findings
- Dark / light theme toggle
- Fully type-safe frontend and backend

---

## Architecture

```
VantagePoint/
├── frontend/       # React 19 + TanStack Router + Tailwind CSS (Vite)
├── backend/        # Node.js + Express + TypeScript  (API proxy / BFF)
└── prediction/     # Python + FastAPI + scikit-learn (ML inference server)
```

**Request flow:**

```
Browser → Frontend (port 3000)
              ↓
       Backend API (port 3006)
              ↓
    Python ML Service (FastAPI)
```

## ML Model

- **Algorithm:** Random Forest Classifier
- **Dataset:** [Student Performance on an Entrance Examination (Kaggle)](https://www.kaggle.com/datasets/adilshamim8/student-performance-on-an-entrance-examination)
- **Target classes:** Excellent (3), Very Good (2), Good (1), Average (0)

**Input features:**

| Category       | Features                                                 |
| -------------- | -------------------------------------------------------- |
| Demographic    | Gender, Caste, Medium of instruction                     |
| Academic       | Class X board & percentage, Class XII board & percentage |
| Socio-economic | Father's occupation, Mother's occupation                 |
| Preparation    | Coaching type, Daily study time                          |

**Evaluation metrics (test set):**

| Metric    | Score  |
| --------- | ------ |
| Accuracy  | 44.78% |
| Precision | 45.10% |
| Recall    | 44.78% |
| F1 Score  | 43.64% |

## Getting Started

### Prerequisites

- Node.js ≥ 18
- Python ≥ 3.10
- A trained model file `student_performance_model.pkl` in `prediction/` (generated by running the notebook)

---

### 1. Train the model (first time only)

```bash
cd prediction
pip install -r requirements.txt
jupyter notebook notebook.ipynb   # run all cells to produce the .pkl file
```

### 2. Start the ML service

```bash
cd prediction
uvicorn main:app --reload --port 8000
```

### 3. Start the backend

```bash
cd backend
npm install
npm run dev
```

### 4. Start the frontend

```bash
cd frontend
npm install
npm run dev
```

The app will be available at **http://localhost:3000**.
